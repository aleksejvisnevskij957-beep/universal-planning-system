УНИВЕРСАЛЬНЫЙ ГРАФИК ПЛАНИРОВАНИЯ ПРОИЗВОДСТВА — ИНСТРУКЦИЯ

УРОВНИ ПЛАНИРОВАНИЯ (от частного к общему):
  1) Операция: дата старта + длительность в сменах (можно дробно, напр. 3,6) — раскладывается по рабочим дням.
  2) Заказ на участке: до 8 операций заказа, строка в форме заказов участка.
  3) Участок: до 30 заказов, график загрузки в % от ёмкости участка (бригад/смену = 100%).
  4) Производство: лист «Мастер» сводит старт/финиш всех заказов по всем 10 участкам,
     «Мастер-график» показывает полный цикл: от разработки КД до сдачи на склад ГП.

СТРУКТУРА ФАЙЛА:
  • «Настройки» — стартовая дата месяца, 10 участков: названия, сотрудники, размер бригады.
    Переименуйте участок здесь — обновится во всех листах, формах и графиках.
  • «Реестр заказов» — единый список до 30 заказов. Номер вводится ОДИН раз и
    автоматически появляется в формах всех участков и в мастере.
  • «У1..У10 Заказы» — ФОРМЫ ЗАКАЗОВ участков (жёлтые ячейки — ввод): по каждому заказу
    8 операций × (Старт / Смен / Факт). Названия операций редактируются прямо в шапке формы.
  • «У1..У10 Ввод» — разворот формы в 240 строк. Заполняется сам, ничего не вводить.
  • «У1..У10 Расчёт» — служебная матрица день × операция. Не менять.
  • «У1..У10 График» — график загрузки участка. В Excel — сводный (итого+лимит),
    в Google скрипт строит полный стековый с операциями и статусными цветами.
  • «Мастер» — свод: старт и финиш каждого заказа на каждом участке + полный цикл в днях.
  • «Мастер-график» — гант полного цикла по заказам (строится скриптом).

ЛОГИКА РАСЧЁТА (единая для всех участков):
  • Ёмкость участка = Сотрудников ÷ Размер бригады (напр. 9÷2 = 4,5 бригады/смену = 100%).
  • Операция длительностью 3,6 смены со старта 09.07: 09.07, 10.07, (Сб-Вс пропуск), 13.07, 14.07 (0,6).
  • План завершения считается с учётом выходных автоматически.
  • Статусы: есть Факт — ✔ Выполнен (зелёный); План-финиш прошёл без факта — просрочен (красный).

УСТАНОВКА В GOOGLE ТАБЛИЦАХ:
  1. Импортируйте этот файл: Файл → Импорт → Загрузить → «Заменить таблицу».
  2. Расширения → Apps Script → вставьте код с вкладки «Скрипт Google» → сохраните.
  3. Запустите функцию setupAll (разрешите доступ). Появится меню «📊 Планирование».
  4. Скрипт построит 10 стековых графиков участков и живой Мастер-гант.
  5. Дальше всё обновляется само при правках форм заказов и при открытии файла.
